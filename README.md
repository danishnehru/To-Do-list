 <h1>📝 To-Do List </h1>

  A simple To-Do List project built using <strong>HTML</strong> to understand basic web structure.

  <h2>🚀 Overview</h2>
    <p>This project demonstrates:</p>
    <ul>
        <li>HTML structure</li>
        <li>Forms and input fields</li>
        <li>Displaying tasks using lists</li>
        <h2>💡 Usage</h2>
<ul>
    <li>Enter a task in the input field</li>
    <li>Click <strong>Add</strong> to save it</li>
    <li>Click <strong>Delete</strong> to remove a task</li>
    <li>Click on a task to mark it as completed</li>
</ul>
